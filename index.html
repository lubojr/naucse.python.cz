7th Python beginners course - Spring 2024

We are planning to open a 7th run of online Python programming beginners course in English starting in Spring 2024, running for 13 weeks. It is aimed for complete beginners without any previous experience in programming. We will go through basic Python syntax, data types and structures and we will work on two projects along the way. The course will be run for two hours each week in a virtual classroom for around 25 attendees and several mentors.

Who are PyLadies?

PyLadies is an international mentorship group with a focus on helping more women become active participants and leaders in the Python open-source community. Our mission is to promote, educate and advance a diverse Python community through outreach, education, conferences, events and social gatherings.

For whom?

Our most important activity in PyLadies Vienna are online semestral Python beginners courses. They will lead you through complete Python programming fundamentals in an entertaining way. No boring math, all lessons are taught on easily imaginable examples and even programming of simple games.

What do you need?

In order to start, you need only motivation, enough free time and your own computer on which you can start to learn programming - with permission to install new software.
